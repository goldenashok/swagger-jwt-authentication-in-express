# swagger-jwt-authentication-in-express
Implemented Swagger with JWT token globally in express api

Implemented Below Feature
- Cors with whitelist option
- openapi
- swagger for get, post, put, delete
- swagger for JWT token
- dotenv feature
- JWT (json web token)
- nodemon feature
